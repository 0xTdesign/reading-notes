What is Wireframe

This is the starting point before you even type one bit of code. First you will need a detailed plan on how your website or the clients website end look and UX experiance layout will be. This can be done either with Software or hand drawn. It is the starting blocks of creating the website. Without a solid frame you will have nothing to reference of.

Key things to consider before wireframing

  • Wireframe > Interactive Prototype > Visual > Design
  • Sketch > Code
  • Sketch > Wireframe > Hi-Def Wireframe > Visual > Code
  • Sketch > Wireframe > Visual > Code

    • Here are a few steps to follow

  • Doing your research what type of user you will be targeting
  • After all data is collected. Consolidate all the data down to quick refereance notes to help
  • Making sure that your user flow is mapped out. This will help you understand the layout and screens pages and the general flow of the website.
  • When designing your wire frame dont think about colours or fonts. This should just be the design layout
